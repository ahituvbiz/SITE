# pensya.info — אתר אחיטוב תכנון פיננסי

האתר של **איתן אחיטוב — יועץ פנסיוני בעל רישיון משרד האוצר**.
תכנון פיננסי הוליסטי לשכירים מעל הממוצע ולעצמאים בני 27-47.

## ארכיטקטורה

- **קבצים סטטיים** — HTML/CSS/JS פשוטים, ללא Build, ללא dependencies.
- **Hosting** — Cloudflare Pages, מתעדכן אוטומטית מ-`main`.
- **כל קובץ HTML עומד בפני עצמו** — CSS ו-JavaScript inline, פונטים מ-Google Fonts CDN.

## מבנה הפרויקט

```
.
├── index.html       דף הבית
├── eitan.jpg        תמונה אישית (לסקציית אודות עתידית)
└── README.md        קובץ זה
```

## עדכון האתר

לעדכון תוכן: עורכים את ה-HTML, מבצעים `git commit` ו-`git push` ל-`main`.
Cloudflare Pages מזהה את ה-push ופורס תוך כ-30 שניות.
